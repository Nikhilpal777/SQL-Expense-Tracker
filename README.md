# SQL Expense Tracker

## Project Overview

This project is a SQL-based Expense Tracker system that allows users to store, manage, and analyze daily expenses.

## Features

* Store and manage expenses
* Filter and sort expense data
* Perform data analysis using SQL
* Categorize expenses using CASE statements
* Use subqueries for advanced insights
* Add expenses using stored procedures
* Automatically log entries using triggers

## Concepts Used

* SQL Queries (SELECT, WHERE, ORDER BY)
* Aggregation (SUM, AVG, COUNT)
* GROUP BY & HAVING
* CASE Statements
* Subqueries
* Stored Procedures
* Triggers

## Technologies

* MySQL

## Sample Insights

* Total spending calculation
* Category-wise expense analysis
* Highest and lowest expenses

## Outcome

Developed strong SQL skills including data manipulation, analysis, and database logic implementation.
